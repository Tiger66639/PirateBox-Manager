Welcome to PirateBox Manager

This is a project that is meant to simplify the entire PirateBox process for laptop users. And after setup phase eventually allow management of files and the "Box" itself.

Authors and Contributors

Cale (@terrorbyte) Black for the management and creation script itself. Matthias (@MaStr) Strubel for making the actual packages that my script uses and David Darts for the piratebox itself.
